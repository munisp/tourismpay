# Schema Patterns

Common database schema patterns used in the 54Link POS platform.

## Core Tables

- **agents** — Agent registration and profile data
- **transactions** — Financial transaction records
- **disputes** — Transaction dispute records
- **settlements** — Settlement cycle records
- **float_balances** — Agent float tracking
- **audit_log** — System audit trail
- **fraud_alerts** — Fraud detection and scoring alert records
- **commission_tiers** — Commission rate tier definitions
